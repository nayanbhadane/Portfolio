# Portfolio Website

Modern, interactive portfolio built with React and Tailwind CSS. The UI emphasizes clean typography, glassmorphism, and smooth motion for a premium presentation of skills and projects.

## Features
- Smooth page load and scroll reveal animations
- Glassmorphism sections with gradient accents
- Scroll progress indicator and custom cursor
- 3D tilt project cards with hover overlays
- Skill tag cloud layout
- Lenis-powered smooth scrolling

## Tech Stack
- React + Vite
- Tailwind CSS
- Framer Motion
- tsParticles
- Lenis
- react-intersection-observer
- react-parallax-tilt

## Link : https://nayan-bhadane-portfolio.vercel.app/

## Setup

Install dependencies:
```bash
npm install
```

Start the dev server:
```bash
npm run dev
```

Build for production:
```bash
npm run build
```

Preview the production build:
```bash
npm run preview
```
